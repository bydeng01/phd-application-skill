## Description

One fully-funded, full-time PhD position with Dr. Viacheslav Borovitskiy's new
research group at the School of Informatics, University of Edinburgh. The group
explores how the **geometry of data** can build better uncertainty-quantifying
models, particularly for structured data types like molecules and images, with
other directions in **geometric learning** available. The advertised description
explicitly covers "learning on structured domains (e.g., graphs, manifolds)" and
exploiting data symmetries — i.e. core geometric deep learning / graph-structured
learning.
